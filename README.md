# Statistics_file

Project Overview

The Petrol Consumption Analysis Dashboard is an interactive data analysis application developed using Python and Streamlit. The project analyzes petrol consumption data through statistical methods and visualizations to help users understand trends and patterns in fuel usage.

Objectives

•	Analyze petrol consumption data.

•	Perform exploratory data analysis (EDA).

•	Identify missing values and outliers.

•	Generate statistical summaries.

•	Visualize data using interactive charts.

Technologies Used

•	Python

•	Streamlit

•	Pandas

•	NumPy

•	Matplotlib

•	Seaborn

•	Scikit-learn

•	SciPy

Features

•	Interactive dashboard interface

•	Dataset preview and summary statistics

•	Missing value detection

•	Correlation heatmap

•	Histograms, box plots, scatter plots, and pair plots

•	Distribution analysis and variance calculation

Conclusion

This project demonstrates how data analysis and visualization techniques can be used to gain meaningful insights from petrol consumption data. The Streamlit dashboard provides a simple and user-friendly interface for exploring the dataset and understanding key statistical patterns.
